## Keyboard Shortcuts

  - F7 = Reveal layer
  - F8 = Reveal comp (not selected layers--the comp you're actually in)
  - ctrl+shift+alt+Q = reset workspace
  - ctrl+shift+alt+W = toggle hold keyframes
  - ctrl+shift+alt+R = run script
  - ctrl+shift+alt+D = run most recent script
